ພາສາ c: c plusplus c++
=================
ພາສາ c ເປັນພາສາ ໂປຣແກຣມມີງທົ່ວໄປ ທີ່ນຳໃຊ້ ໃນການພັດທະນາ ຊອບແວ ໄດ້ຫລາຍຮູບແບບ.  
ມາຈາກພາສາ BCPL (Basic Combined Programming Language) ແລະ ພາສາ B.  ພັດທະນາຂື້ນໃນປີ 1967 ໂດຍ ມານຕິນ ລີຊາດ (Martin Richard) ແລະ ທ່ານ ເດນນິສ ຣິທຊີ (Dennis Ritchie) ໃນຊ່ວງປີ 1971 ເຖີງ 1973 ໄດ້ເພີ່ມຊະນິດຂໍ້ມູນ ແລະ ຄວາມສາມາດອື່ນໆ ໃຫ້ຫຼາຍຂື້ນ ແລະ ນໍາພາສາ C ໄປພັດທະນາລະບົບປະຕິບັດການ ຢູນິກ(Unix) ໃນເຄື່ອງຄອມພິວເຕີ, ພາສາ C   ເປັນທີ່ນິຍົມໃຊ້ຢ່າງຫຼວງຫຼາຍໃນຊ່ວງປີ 1980 ຈົນມີຄວາມພະຍາຍາມກໍານົດມາດຕະການຂອງພາສາເພື່ອໃຫ້ສາມາດໃຊ້ ພາສາ C ໄດ້ໃນຄອມພິວເຕີອື່ນໆ ໃນປີ 1983 ໂດຍ ANSI(The American National Standards Institute)ໄດ້ຕັ້ງຄະນະກຳມະການ X3J11 ເພື່ອສ້າງມາດຕະການດັ່ງກ່າວ ແລະ ໄດ້ຮັບການກວດສອບ  ແລະ ຍອມຮັບໂດຍ ANSI ແລະ ISO(The International Standards Organization) ໂດຍມີການພິມມາດຕະການຂອງພາສາ  C ໃນປີ 1990 ຈາກຄວາມມີປະສິດທິພາບ ແລະ ຄວາມສາມາດທຳງານໃນຄອມພິວເຕີອື່ນໆ. ຈາກນັ້ນພາສາ C ຈື່ງໄດ້ມີການນຳໄປພັດທະນາລະບົບປະຕິບັດການຕ່າງໆ ແລະ ໃຊ້ເປັນຕົ້ນແບບຂອງພາສາອື່ນໆ ທີ່ສຳຄັນໃນປະຈຸບັນ ຄື: ພາສາ C++, Java..... ເປັນຕົ້ນ.

- ຖ້າຄະແນນ =4 ໃຫ້ບອກວ່າ "Grade A";
- ຖ້າຄະແນນ >4 ໃຫ້ບອກວ່າ "Error";



#include <iostream>
using namespace std;
int main()
{
    char name[20];
    int age;
    float s;
    char grade;
    cin>>name;
    cin>>age;
    cin>>s;
    if(s<1)
        cout<<"Grade F"<<endl;
    else if(s>=1 && s<=1.9)
        cout<<"Grade D"<<endl;
    else if(s>=2 && s<=2.9)
        cout<<"Grade C"<<endl;
    else if(s>=3 && s<=3.9)
        cout<<"Grade B"<<endl;
    else if(s<=4)
        cout<<"Grade A"<<endl;
    else
        cout<<"Error"<<endl;
    cout<<name<<age<<grade<<s;
}

24. ຈົ່ງວາງຕົວປ່ຽນ day  ນໍາໃຊ້ Switch ເພື່ອສະແດງ ວັນເສົາ ແລະ ວັນອາທິດ ຖ້າບໍ່ແມ່ນວັນ ເສົາ ແລະ ອາທິດ ໃຫ້ມັນບອກວ່າ "ລໍຖ້າໃຫ້ຮອດທ້າຍອາທິດ".


#include <iostream>
using namespace std;
int main() {
  int day = 4;
  switch (day) {
    case 6:
      cout << "Today is Saturday";
      break;
    case 7:
      cout << "Today is Sunday";
      break;
    default:
      cout << "Looking forward to the Weekend";
  }
  return 0;
}

25. ຈົ່ງວາງຕົວປ່ຽນ g ເພື່ອປ້ອນ ຄ່າ  A,B,C ໂດຍນຳໃຊ້ Switch 
	ຖ້າປ້ອນ A ໃຫ້ບອກວ່າ "Excellent";
	ຖ້າປ້ອນ B ໃຫ້ບອກວ່າ "Very Good";
	ຖ້າປ້ອນ C ໃຫ້ບອກວ່າ "You Passed";
	ຖ້ານອກຈາກນັ້ນ ໃຫ້ບອກວ່າ "Invalid Entry";

#include <iostream>
using namespace std;
int main()
{
    char g;
    cin>>g;
    switch(g)
    {
    case 'A': cout<<"Excellent"; break;
    case 'B': cout<<"Very Good"; break;
    case 'C': cout<<"You Passed"; break;
    default: cout<<"Invalid Entry"; break;
    }
}
